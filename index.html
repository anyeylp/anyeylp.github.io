Posted on 2018-09-12

hexo 命令行

hexo new “文章名”: 新建文章
hexo g(generate): 生成网站静态文件到默认设置的public文件夹
hexo d(deploy): 自动生成网站静态文件，并部署到设定的仓库
hexo s(server): 启动本地服务器，用于预览。 默认地址： http://localhost:4000/
hexo clean :清除缓存文件 db.json 和已生成的静态文件public。

命令行使用顺序

hexo clean
hexo g
hexo d
hexo s
